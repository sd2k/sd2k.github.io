Grafana Live features, providing streaming functionality to Grafana.

The types in this module are used by Grafana Live, a real-time messaging engine introduced in Grafana v8.0. Grafana Live allows you to push event data to a frontend as soon as the event occurs. Generally this will be done by plugins that implement the StreamService trait.

See the Grafana Live documentation for more information.
